# Polymarket SDK – Examples

This directory contains helper files and scripts for working with the SDK examples.

## Files

- `.env.example` – a sample set of environment variables. Copy it to `.env` and fill in values for your own setup.
- `print-env.ts` – a script that prints the status of the main environment variables and helps you verify that `.env` is configured correctly.

## Quick start

1. Copy `.env.example`:

   ```bash
   cp examples/.env.example examples/.env
 ```
